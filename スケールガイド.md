# ToyLib スケールガイド（1ユニット = 1メートル）

## 基本的なオブジェクトサイズ

| オブジェクト             | 推奨サイズ（m）            | 備考 |
|--------------------------|-----------------------------|------|
| プレイヤー（ヒーロー）     | 高さ: 1.7〜1.8             | 肩幅0.5〜0.6程度 |
| 敵（小型）               | 高さ: 1.0                  | スライム等 |
| 敵（中型）               | 高さ: 2.0                  | ゴーレムなど |
| 敵（大型）               | 高さ: 4.0〜8.0             | ボス向け |
| 段差（登れる）           | 高さ: 0.3〜0.6             | ジャンプ可能 |
| 段差（登れない）         | 高さ: 1.0以上              | スキルやアクションで登る |
| ドア                     | 幅: 1.2〜1.5、高さ: 2.0〜2.5 | 建物用 |
| アイテム（小）           | 0.2〜0.5                   | コイン、回復アイテムなど |
| アイテム（大）           | 1.0〜2.0                   | 宝箱など |

## UI スケール（ミニマップ / HPバーなど）

### ミニマップ
- 表示範囲: 20〜50m 四方
- プレイヤーアイコン: 直径 0.2〜0.3m
- 敵アイコン: 直径 0.1〜0.2m
- 配置: 画面右上 or 左上

### HP・スタミナバー
- HPバーサイズ: 幅 200〜300px、高さ 20〜30px
- スタミナゲージ: 幅 150〜250px、高さ 10〜20px

---

## カメラ設定パラメータ（ビュー & プロジェクション）

### ビュー行列（View Matrix）

#### TPS視点（肩越しカメラ）
- カメラ位置: プレイヤー位置 + (0, 5, -10)
- 注視点: プレイヤー位置 + (0, 1.5, 0)
- 上方向: (0, 1, 0)

#### FPS視点（目線カメラ）
- カメラ位置: プレイヤー位置 + (0, 1.6, 0)
- 注視点: カメラ位置 + 前方ベクトル
- 上方向: (0, 1, 0)

---

### プロジェクション行列（Perspective）

| パラメータ         | 推奨値       | 備考 |
|--------------------|--------------|------|
| FOV（視野角）       | 60°〜70°     | ラジアンで約1.0〜1.2 |
| アスペクト比        | 画面幅 ÷ 高さ | 例: 16:9 = 1.777... |
| Near Clip          | 0.1〜0.3     | 小さすぎると精度低下 |
| Far Clip           | 100〜500     | 背景まで描画する場合は拡張可 |

---
